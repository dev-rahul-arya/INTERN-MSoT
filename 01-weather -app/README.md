## Project Info

A weather application built with modern web technologies to provide real-time weather forecasts and detailed meteorological information.

## How to Get Started

### Using GitHub

Clone this repository and start developing locally with your preferred IDE.

**Requirements:**
- Node.js & npm ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

**Setup steps:**

```sh
# Step 1: Clone the repository
git clone <https://github.com/dev-rahul-arya/INTERN-MSoT.git>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

The app will run at `http://localhost:5173` with hot module reloading enabled.

## Tech Stack

This project uses:

- **Vite** - Fast build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - High-quality component library
- **Tailwind CSS** - Utility-first CSS framework
